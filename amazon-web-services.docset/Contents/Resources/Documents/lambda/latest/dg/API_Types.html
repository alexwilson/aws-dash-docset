  • DeadLetterConfig

  • Environment

  • EnvironmentError

  • EnvironmentResponse

  • EventSourceMappingConfiguration

  • FunctionCode

  • FunctionCodeLocation

  • FunctionConfiguration

  • TracingConfig

  • TracingConfigResponse

  • VpcConfig

  • VpcConfigResponse
