Amazon Redshift Parameter Type

Describes parameters for the AWS::Redshift::ClusterParameterGroup resource type.

Syntax

JSON

{
  "ParameterName" : String,
  "ParameterValue" : String
}

YAML

ParameterName: String
ParameterValue: String

Properties

ParameterName

The name of the parameter.

Required: Yes

Type: String

ParameterValue

The value of the parameter.

Required: Yes

Type: String
